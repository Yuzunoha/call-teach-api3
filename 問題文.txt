[SNS-API課題]
バッチさんが作ったSNSのAPIを叩きまくる課題です。0期生が元気よく叩きまくっていました。

リクエストヘッダには Content-Type: application/json を含めます。
また"headerにtoken入れる"という注釈があるAPIは、リクエストヘッダにベアラートークンを含めることが必須のAPIです。
ベアラートークはリクエストヘッダに下記のように含めます
Authorization: Bearer XXXXXXXXXXXXXXXXXXXXXXXXXXXXX

ベアラートークンはユーザ登録やユーザログインのAPIを叩くとレスポンスで取得できます
ベアラートークンはローカルストレージに保存しましょう(ネット上で賛否はありますが)

[ユーザー登録]
POST: https://teachapi.herokuapp.com/sign_up
パラメーターが
body parameterで
```
{
    sign_up_user_params: {
      name: name,
      bio: bio,
      email: email,
      password: password,
      password_confirmation: passwordConfirmation
    }
}
```
[ユーザーログイン]
POST: https://teachapi.herokuapp.com/sign_in
パラメーターが
body parameterで
```
{
    sign_in_user_params: {
      email: email,
      password: password,
      password_confirmation: passwordConfirmation
    }
}
```
[ユーザー一覧]
GET https://teachapi.herokuapp.com/users
パラメーターが
query parameterで
```
page: ページの番号(数字)
limit: 1ページあたり何個あるか
query: 単語を入れると検索できる
headerにtoken入れる
```
[ユーザー編集]
PUT https://teachapi.herokuapp.com/users/{id}
body parameterで
```
{
    user_params: {
      name: name,
      bio: bio,
    }
}
```
headerにtoken入れる
[ユーザー削除]
DELETE https://teachapi.herokuapp.com/users/{id}
headerにtoken入れる
